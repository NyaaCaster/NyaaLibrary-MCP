### 关键实现要点

- **维度守卫**：POST memory 在 `writeMemory` 前先调 `dimensionInSync()`，不一致直接 409 拒写，避免触发 `ensureMemVecTable` 的 DROP 分支
- **ID 校验**：`Number(req.params.id)` 非正整数时 400，不传 `NaN` 给 service
- **zod 校验**：`postMemorySchema` + `putProfileSchema`，对标 `kb.ts` 风格
- **鉴权隔离**：ownersRouter 挂在 `requireSession` 之后，与猫猫链路 `/api/memory/*` 完全隔离
- **错误透传**：`writeMemory` 抛错时 500 + 原文，前端可直接展示
