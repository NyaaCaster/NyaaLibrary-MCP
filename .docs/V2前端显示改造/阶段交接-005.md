### 关键行为

| 操作 | 实现 |
|------|------|
| 列表 | `useQuery(["owner-memory", ownerKey])` → 全量拉取，前端 useSort + Pagination（PAGE_SIZE=10） |
| 搜索 | 前端按 `content.toLowerCase().includes(filter)` 过滤 |
| 排序 | 可排序列：内容摘要 / 字符数 / Salience / 创建时间（默认 created_at DESC） |
| 查看 | Eye 按钮 → MemoryDetailModal（readonly textarea + 元信息） |
| 删除 | MemoryDetailModal 内「删除记忆条目」→ 二次确认 → DELETE …/memory/:id → invalidate ["owner-memory"] + ["owners"] |
| 补充 | 「补充记忆」按钮 → MemoryCreateModal（多行文本 + 可选 salience）→ POST …/memory → 成功刷新、失败红字提示 |
